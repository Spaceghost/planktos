Planktos

Serving websites over bittorrent

About Planktos

Planktos is a library for enabling a website's static assets to be served over BitTorrent. By utilizing service workers and WebTorrent, a website can be served from the users of the site instead of from a server. Using Planktos allows site owners to: offset declining ad revenue through reduce bandwidth, increas scalability since download speeds increase with more users, and increase uptime since static content can still be retreived from BitTorrent even though the server is down. The only setup required is creating a torrent with the static content and including a script tag in the web pages of the site. No extra software or browser plugins needed; Planktos works in plain Chrome, Firefox, and Opera. If the user's browser doesn't support Planktos then everything fallsback nicely to regular https.

Install Planktos

Installing planktos is as simple as registering the Planktos service worker, and generating the torrent.
