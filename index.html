Planktos

Serving websites over bittorrent

Planktos enables websites to serve their static content over bittorrent from the users of the website

peer-to-peer

Users download the static content from other users with webtorrent. This allows site owners to reduce bandwidth usage and offset declining ad revenue.

Fast & Scalable

With more users, downloads get faster.

Secure

All peer downloaded files are verified with sha1 as required by the bittorrent spec.

Simple & Easy Integration

The Planktos service worker intercepts HTTP GET requests and retreives the files over bittorrent. No code changes required other than the one liner to register the service worker.

Supports all files

Whether it's html, css, javascript, image, or video files it all works with Planktos.

Supports top browsers

Planktos supports Chrome and Firefox with no extra software or plugins needed. Other browsers are working to implement the APIs Planktos requires like webrtc and service workers.

Improved Caching

Once files are downloaded, they are persisted to disk and used for any subsequent requests. There is no need for browsers to ask the web server if files have changed because the torrent stores the hashes of those files.

Avoid downtime

Planktos is still able to download files from bittorrent even if the website's servers go down, and users can still view already downloaded content even if they are offline.

Install Planktos

Installing planktos is as simple as registering the Planktos service worker and generating the torrent.
